CMYK Drawingtylerchen

This project was inspired by the process inkjet printers use to create the appearance of a range of colors using just four discrete colors. I used GIMP to decompose an image of a bird into four images, each giving values of one of the four colors I would be working with. I then made four 18x24in marker drawings, one for in each color. Various values are created by varing the size of the dots.


Below are black and white scans of the Cyan, Magenta, Yellow and Key drawings of the bird. I then used GIMP to combine and colorize these images and merge them into the composite image seen above.
