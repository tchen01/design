LED Lightingtylerchen

I used the fadecandy board and drivers to run my lighting.

I wrote python code to create a variety of ambient fading effects on my light strips. There are a variety of functions and wrappers which allow various color schemes and effects to be easily displayed on the LED strips. Some care was required for the hexagon, so that the fading wraped around smoothly.

The below image is a screengrab from the Netflix comedy "Master of None" which insipred the color scheme in the above image. Other color schemes include one titled summer which is inspired by colors of the sunset out I saw out my window while working on this project.

The hexagon was created by first making a hexagon frame, cutting LED srips to fit on the farme, and finally soldering them back together.
